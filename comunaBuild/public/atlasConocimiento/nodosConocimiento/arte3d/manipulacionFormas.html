Crear permite generar formas basicas como cubos, esferas, cilindros, etc.

Para agregar un mesh a la escena, se puede agregar.

  • Dirigiendoce al boton ADD, donde aparecera un listado bastante amplio de mesh.
  • Uno de los atajos para la generacion de un mesh puede ser el Shitf + A.
Eliminar sirve para eliminar objetos o partes de un objeto selenccionado.

Para eliminar un mesh, se puede hacer:

  • Presionando X, al tener el cursor en el viewport y Presionar x se abrira un menu contextual, en este menu se encontraran multiples opciones Pero nos centraremos en las basicas como estas: vertices, bordes, caras
 Mover permite desplazar objetos en el espacio 3D.

Para mover un objeto, se puede hacer:

  • Presionando en la barra de herramientas del viewport estara la opcion de mover objeto, al precionar aquella opcion apareceran unas flechar de colores, cada color significara un eje, el rojo representa el eje X, el verde representa el eje Y y el azul representa el eje Z, al hacer click sobre ellas y desplazar un poco el mause se puede observar como el objeto se mueve de un lado a otra o en la direccion que se le haya asignado.
  • Para asignarle una direccion de movimineto a un objeto atravez de atajos se utiliza:

    G (Movimineto libre)

    G + Z (Movimiento en el eje Z)

    G + X (Movimiento en el eje X)

    G + Y (Movimiento en el eje Y)

Escalar permite ajustar el tamaño de los objetos.

Para escalar un objeto se puede hacer con:

  • Presionando la barra de herramientas del viewport estara la opcion scale, al presionar este boton aparecera unas flechas pero esta vez las puntas seran cubos, al presionar y arrastrar se podra escalar el objeto dependiendo de el eje.
  • Uno de los atajos para cambiar el tamaño de un objeto es la tecla S, al presionarla se podra cambiar el tamaño del objeto, al mover el mause se cambiara el tamaño en general y al presionar las teclas X, Z y Y su tamaño cambiara en un eje determinado.
Rotar permite girar objetos alrededor de un eje.

para rotar un objeto se puede hacer con:

  • Presionando la barra de herramientas del viewport estara la opcion rotate, al presionar este boton aparecera una esfera al rededor del objeto con lineas de rotacion correspondientes a cada eje, al presionar y arrastrar estas lineas se podra rotar el objeto dependiendo de el eje que se seleccione.
  • Uno de los atajos para cambiar el eje de rotacion del objeto vendria siendo la R, al presionarla se podra rotar el objeto, al mover el mause el objeto rotara libremente y al presionar las teclas X, Y y Z su rotacion cambiara al eje determinado.
Dupicar permite crear copias identicas de objetos seleccionados.
  • Estando en modo objeto, se hace click derecho sobre el objeto para abrir el menu contextual, y se presiona Duplicate object.
  • Uno de los atajos para duplicar es presionando Shift + D,despues de esto el objeto se quedara en la misma posicion del objeto que hemos duplicado asi que habra que moverlo a algun lugar de la escena.
Seleccionar permite elegir objetos o partes especificas de un objeto para realizar acciones sobre ellos.
  • Para seleccionar un objeto se debe hacer click izquierdo encima de este, o presionando la tecla A para seleccionar todo.
  • Para deseleccionar un objeto se hace click izquierdo en una zona vacia de la escena, o presionando dos veces A.
